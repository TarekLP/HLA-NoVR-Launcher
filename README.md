# HLA-NoVR-Launcher - Avalonia
Launcher for Half-Life: Alyx NoVR remade in Avalonia UI with .net 10.0

## Installation

### Windows
[Download the latest release.](https://github.com/HLANoVR/HLA-NoVR-Launcher/releases/latest/download/HLA-NoVR-Launcher.exe)
Extract the Launcher(exe-file) into a folder, then you start it. You'll see it create a bunch more files (dont worry about those.).
Once the Launcher has booted you select the Half-Life: Alyx Installation Folder (it's located where Steam installed the game and it has the folders `game` and `content` inside of it).
Afterwards you can just click play, watch it install and then when everything goes well you'll just see the game menu where you can start a new game.

---

### Steam Deck/Linux

If you use a Steam Deck/Linux, see the [FAQ](https://docs.google.com/document/d/1mlDz24iE1r4Lf16y5N9I37ZIvm4V0ie2Sxg1GBlcs10) for installation instructions.

## Important Information

In case you need to select a folder after clicking "Play" in the launcher, select your "Half-Life Alyx" folder (it's located where Steam installed the game and it has the folders `game` and `content` inside of it).
Only official copies of the game bought on Steam are supported.
